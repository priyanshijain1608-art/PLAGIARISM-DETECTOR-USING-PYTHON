# PLAGIARISM-DETECTOR-USING-PYTHON
A simple and efficient plagiarism detection tool built using Python that compares two text files and calculates the similarity percentage between them. This project uses the difflib library to analyze textual data and identify matching content, making it useful for students, educators, and content creators to detect duplicate or copied material.
# Plagiarism Detector using Python

This project detects similarity between two text files using Python.

## 📌 Features
- Compares two text files
- Shows similarity percentage
- Simple and easy to use

## 🛠️ Technologies Used
- Python
- difflib library

## ▶️ How to Run
1. Add two text files (txt1.txt and txt2.txt)
2. Run the Python script
3. Get plagiarism percentage

## 📊 Output Example
The plagiarized content is 75.34%
